# Sorter Game - Required PNG Sprites

Place these 5 PNG files in this folder (`assets/sorter/`):

| File | Description | Usage |
|------|-------------|-------|
| **seed.png** | White/light seed icon | Base for plant seeds. Phaser tints it green, yellow, or red based on type. |
| **flower.png** | White/light flower icon | Base for grown plants on the field. Same tint colors (green, yellow, red). |
| **consumer.png** | White/light consumer icon | Dedicated sprite for consumers (who wants which plant). Tinted green, yellow, or red. |
| **crate.png** | Wooden crate/box icon | Used as-is for crates (no tint). |
| **water.png** | Water drop/bottle icon | Used as-is for water supply (no tint). |

## Recommended specs
- **Size:** 64x64 px or 128x128 px (will be scaled)
- **Format:** PNG with transparency
- **Style:** Simple, flat, white or light gray shapes on transparent background (for tinting)

## Color reference for seed/flower/consumer tints
- **Green:** `#2ecc71` (0x2ecc71)
- **Yellow:** `#f1c40f` (0xf1c40f)
- **Red:** `#e74c3c` (0xe74c3c)
