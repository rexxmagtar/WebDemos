# Balloon Arrows - Required Sprites

Place this PNG file in this folder (`assets/balloon-arrows/`):

| File | Description | Usage |
|------|-------------|-------|
| **balloon.png** | White/light balloon shape | Balloons in queues and on conveyor. Phaser tints it red, blue, or yellow. |

## Recommended specs
- **Size:** 64×64 px or 128×128 px (displayed at ~32×32 px)
- **Format:** PNG with transparency
- **Style:** Simple balloon shape (round/circle) in white or light gray on transparent background – needed for tinting

## Color reference for tints
- **Red:** `#e74c3c` (0xe74c3c)
- **Blue:** `#3498db` (0x3498db)
- **Yellow:** `#f1c40f` (0xf1c40f)
